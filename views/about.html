About TaskManager

Project Overview

TaskManager is a simple and intuitive web application designed to help users organize, track, and manage their daily tasks efficiently. This project is part of our Web Development course assignment, demonstrating practical skills in Node.js, Express.js, and web application development.

Our Mission

Our goal is to create a user-friendly task management system that helps people stay organized and productive. We believe that effective task management is essential for personal and professional success.
